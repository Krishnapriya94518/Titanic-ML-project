# Titanic-ML-project
Titanic survival prediction using machine learning models and exploratory data analysis.

Goal
Predict whether a passenger survived the Titanic disaster.

Dataset
Kaggle Titanic dataset

Tech Stack
Python
Pandas
Scikit-learn
Seaborn
Matplotlib

Models Used
Logistic Regression
Decision Tree
Random Forest

Best Model Accuracy
83%

Key Findings
- Female passengers had higher survival rates
- First-class passengers survived more
- Family size influenced survival probability
